About AWS S3 Simulator

Purpose

This project was built to simulate the core functionality of AWS S3 (Simple Storage Service) in a local, self-contained environment. It allows developers to prototype, test, and understand object storage concepts without incurring cloud costs or needing an internet connection.

What it does

The simulator provides a fully functional S3-compatible API powered by MinIO. It includes:

  • Bucket Management: Create, list, and delete buckets.
  • Object Storage: Upload, view, and manage files.
  • REST API: A documented API for programmatic interaction.
  • Web Dashboard: A user-friendly interface to visualize and manage your storage.

Why it's useful

  • Zero Cost: Run S3 locally for free.
  • Offline Development: Work on your projects anywhere.
  • Education: Great for learning how object storage works under the hood.
  • Testing: Ideal for integration tests in CI/CD pipelines.

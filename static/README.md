# Static Files Directory

This directory contains static assets for the web interface.

## Structure

```
static/
├── css/
│   └── style.css (optional custom styles)
├── js/
│   └── app.js (optional additional JS)
└── images/
    └── (optional images)
```

Currently, all styles are loaded via Tailwind CDN in the HTML template.
Additional custom assets can be placed here if needed.
